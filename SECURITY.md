# Security Policy

PresetSnip is a local desktop utility. It does not upload screenshots, send telemetry, or contact a network service.

## Reporting Issues

Open a security issue or contact the maintainers through the repository's preferred private channel if one is configured.

Please include:

- affected version or commit
- clear reproduction steps
- expected behavior
- observed behavior
- relevant logs or exception details

Do not include private screenshots or sensitive captured content in public issues.
