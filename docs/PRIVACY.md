# Privacy

PresetSnip is local-first.

- It does not upload screenshots.
- It does not use telemetry.
- It does not call external APIs at runtime.
- It stores settings locally under the current user's app data directory.
- Screenshots are saved only to the directory selected by the user.
- Sound is generated locally at runtime; no sound files are downloaded.
- Notifications are local Windows desktop notifications.

The repository includes an AI-generated app icon created for this project. The runtime application does not use image generation.
